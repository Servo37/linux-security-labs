# Permissões no Linux

## 📌 Conceito
Permissões definem quem pode ler (r), escrever (w) e executar (x) um arquivo.

---

## 🔐 Estrutura das permissões

Exemplo:
-rwxr-xr--

Usuário | Grupo | Outros

---

## 💻 Comandos importantes

ls -l
chmod 644 arquivo.txt
chmod 755 script.sh
chown root:root arquivo.txt
