# Usuários e Grupos no Linux

## 📌 Conceito

Linux é um sistema multiusuário.  
Cada usuário pode pertencer a um ou mais grupos.

---

## 👤 Comandos importantes

whoami
id
useradd nome
passwd nome
usermod -aG grupo nome
groupadd grupo

---

## 🔐 Aplicação em Cibersegurança

Gerenciamento correto de usuários evita:

- Escalada de privilégio
- Acesso indevido
- Movimentação lateral
